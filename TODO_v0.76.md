## A. 추가 기능 / 수정 사항

- [x] **A1. 목표 수정 요청 회수** *(완료)*
  PENDING_MODIFY 상태에서 본인이 회수 가능. 진행률 따라 APPROVED 또는 IN_PROGRESS 로 자동 복구.

- [ ] **A2. 주간업무 — 팀 업무 현황 코멘트 수정·삭제**
  팀 업무 현황 페이지에서 작성한 팀 코멘트(`WeeklyTask.leadComments`)를 작성자 본인이 수정·삭제할 수 있게 한다.

- [ ] **A3. 전사 육성면담서 — 직무 요청사항 한글 표기**
  HR관리자 `/mentoring/all` 에서 "직무 요청사항"이 영문(`STAY` 등)으로 표기되는 문제. 한글·동일 용어로 통일.

- [ ] **A4. 육성면담서 제출 후 수정 절차**
  - 제출 완료된 육성면담서는 작성자 본인이 **수정 요청** 가능
  - HR관리자가 **수정 승인** 가능
  - 승인 후 해당 면담서 Lock 해제 → 임시저장/제출 기능 재활성화

- [ ] **A5. 1on1 대상 변경**
  - "새 1on1 대화 시작"의 "팀장 선택" → **"대화 대상"** 으로 용어 변경
  - 직속 팀장·임원 선택 가능, 소속 팀원은 제외

- [x] **A6. 사용자관리 — 비활성화 표기 + 활성화 가드** *(완료)*
  - 비활성화된 사용자가 "초대 대기"로 잘못 표기되는 문제 → "비활성화"로 표기
  - 초대 수락 전 비활성화된 계정이 초대 수락 시 자동 활성화되지 않도록 가드

- [x] **A7. 사용자관리 — 재활성화 오류 수정** *(완료)*
  - 비활성화된 사용자를 재활성화할 때 오류 발생하는 문제 수정
  - 재활성화된 사용자의 기존 기록(목표·평가·면담 등)이 그대로 불러와지는지 검증

- [x] **A8. 핵심목표 책임자 필드** *(완료)*
  - 목표 추가 시 책임자 입력 기능 추가
  - 본인이 등록하는 경우 본인 이름이 자동 입력

- [x] **A9. 골카드 책임자 표시** *(완료)*
  - 골카드에 `책임자 : 이름` 표시
  - 공동추진자가 있으면 `공동` 배지 표시

- [x] **A10. 사용자 삭제 시 목표 자동 이관** *(완료)*
  - 사용자 삭제 → 기존 핵심목표를 팀장에게 이관 (팀장 부재 시 본부장 → 부문/공장 임원)
  - 팀장이 책임자를 재지정 → 해당 책임자 목표로 이동

- [ ] **A11. 전사 육성면담서 — 특이사항 메뉴**
  - 직무 요청사항이 "만족함" 외 항목인 인원 + 작성 내역을 별도 메뉴(특이사항)로 표시

- [ ] **A12. 평가 확정 후 수정 요청**
  - 임원이 팀장·팀원 평가 확정 후, 수정 필요 시 HR관리자에게 수정 요청
  - HR관리자가 해당 건만 "확정 전" 상태로 되돌리는 기능
  - ⚠️ CLAUDE.md §6-1 인사평가 가시성 원칙 관련 — 신중 진행
