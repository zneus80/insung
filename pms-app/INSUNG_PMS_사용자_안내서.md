# INSUNG PMS 사용자 안내서

> 인성 성과·인사관리 시스템(Performance Management System) 기능 안내서
> 조직 단위 평가 등급 설정 → 인원 배정 → 자기평가 → 인사평가 → 평가등급 확정까지 전 과정을 시스템에서 처리합니다.

---

## 1. 시작하기

### 1.1 로그인
- **이메일 / 비밀번호** 로그인 방식입니다.
- 비밀번호를 잊었다면 로그인 화면의 **비밀번호 재설정** 으로 메일을 받아 변경합니다.
- 초대를 받은 신규 사용자는 메일·링크의 **초대 수락** 화면에서 비밀번호를 설정합니다.
  - 비밀번호 정책: **8자 이상 + 소문자 + 특수문자 포함**.
- 보안: 90일이 지난 비밀번호는 변경 안내 배너가 뜨며, 다른 기기에서 비밀번호 변경 시 기존 세션은 강제 로그아웃됩니다. 일정 시간(30분) 미사용 시 자동 로그아웃됩니다.

### 1.2 사이드바와 권한
화면 왼쪽 사이드바에 **본인 역할에 해당하는 메뉴만** 표시됩니다. 메뉴가 많은 경우 그룹(기본정보입력·인사평가 설정·시스템 설정 등)은 **접기/펼치기** 가 가능합니다.

## 2. 역할 정의

| 역할 | 설명 |
|------|------|
| **팀원 (MEMBER)** | 자기 목표 관리·자기평가·육성면담서 작성, 본인 평가결과 확인 |
| **팀장 (TEAM_LEAD)** | 팀원 목표 승인, 팀원 인사평가(1차 의견), 자기평가 작성 |
| **본부장 (HQ_HEAD)** | 본부 산하 팀장·팀원에 대한 인사평가(2차 의견) |
| **차순위 임원 (EXEC_SUB)** | 부공장장/부부문장/부본부장 — 팀장과 동일하게 평가 의견만(확정 불가) |
| **임원 (EXECUTIVE, 최상위)** | 소관 조직 진행현황 확인 + **평가등급 최종 확정** |
| **최고관리자 (CEO)** | 조직별 평가 등급 결정(최종 권한), 전사 현황·AI 분석 |
| **HR관리자 / HR마스터** | 직급에 **추가되는 권한**. 평가기간·인원배정·이력·등급설정·백업 등 관리 |

> 임원 역할은 **조직 배정 단위**로 관리됩니다. 같은 사람이 A조직에서는 최상위 임원, B조직에서는 차순위 임원이 될 수 있습니다.

## 3. 공통 기능 (모든 역할)

### 3.1 대시보드 (`대시보드`)
- 전체목표 / 평균진행률 / 완료 / 승인대기 요약 — 클릭 시 해당 페이지로 이동.
- 진행 중인 목표, 내 마일리지 등 위젯 제공.
- 임원·최고관리자 대시보드: 담당 조직의 팀별·개인별 목표 현황, 회사·조직 경영목표 표시.

### 3.2 알림 (`알림`)
- 승인 요청/처리, 평가 단계 전달, 주간보고 작성 등 업무 알림.
- 보안 알림은 HR마스터·관리자에게만 전달됩니다.

### 3.3 공지사항 (`공지사항`)
- 사내 공지 게시판. 새 공지는 사이드바 배지로 표시됩니다.

## 4. 핵심목표 관리 (팀원·팀장)

### 4.1 핵심목표관리 (`핵심목표관리`)
- 본인 및 팀의 핵심목표(과제업무)를 등록·관리합니다.
- 목표 상태 흐름:
  `임시저장 → 승인요청 → (팀장 1차) → 최종승인 → 진행중 → 완료` / `포기요청 → 포기`
- 목표마다 **공동수행자** 를 지정할 수 있으며, 공동수행자도 해당 목표를 함께 관리합니다.
- 업무 **이관**: 수행자를 변경하면 이전 수행자는 자동으로 빠지고 가중치 슬롯도 정리됩니다.

### 4.2 가중치 배분
- 핵심목표는 **개인별 가중치(%)** 를 가집니다(본인 핵심목표 합 = 100% 기준).
- '가중치 배분' 상신 → 승인 절차를 거쳐 확정됩니다(승인 시점에 소유 여부 재검증).
- 이관 중·재지정 대상 업무는 가중치 배분에서 제외됩니다.

### 4.3 목표 진행현황
- **팀원·팀장**: 본인/팀 목표 진행현황.
- **임원** (`핵심목표 진행현황`): 소관 조직을 **팀 목표별** 로 확인하며, 목표 카드 우측에 수행자 이름·가중치가 표시됩니다. '개인별' 토글로 사람 기준 보기도 가능합니다.
- **최고관리자** (`조직목표현황`): 부문/공장 카드 → 클릭 시 조직 트리가 펼쳐지고, **팀(조직) 업무 기준** 으로 목표가 진행중→완료→포기 순으로 표시됩니다(각 목표에 수행자 이름).

---

## 5. 주간업무보고 (`주간업무보고`)

- 한 주의 **Has Done(완료)** / **Will Do(예정)** 를 기록합니다. 이전 주 Will Do는 다음 주 Has Done으로 자동 이월됩니다.
- 핵심목표에 연동되는 항목은 **해당 목표의 수행자(소유자+공동수행자)만** 추가·수정·삭제할 수 있습니다.
- 핵심업무목표 블록의 표시 순서는 ▲▼ 로 조정 가능합니다.

---

## 6. EGG Meeting — 자기평가 · 육성면담서

### 6.1 자기평가 (`자기평가`, 팀원·팀장)
- 핵심목표(과제업무)와 일반업무에 대한 **자기 의견 + 점수(0~100)** 를 작성합니다.
- 핵심목표는 가중치 환산(핵심 80% 비중)으로 가중평가점수가 계산되고, 총점이 표시됩니다.
- 포기한 핵심업무도 마지막에 포기 배지와 함께 표시됩니다.
- **수정요청**: 상위 권한자의 평가 확정 전까지 HR에게 자기평가 수정요청을 보낼 수 있습니다.

### 6.2 육성면담서 (`육성면담서`, 팀원·팀장)
- 직무정보·경력개발 방향·직무요청·종합의견을 작성합니다.
- 자격증은 항목별 개별 입력, 연도별 이력 관리가 가능합니다.

### 6.3 전사 육성면담서 확인 (`전사 육성면담서 확인`, 최고관리자·HR관리자)
- 전 직원 육성면담서를 조직 기준으로 열람합니다.

## 7. 인사고과 (평가)

### 7.1 평가 프로세스 흐름
```
① 조직평가관리(최고관리자)  →  ② 조직평가인원관리(HR)  →  ③ 자기평가(팀원·팀장)
   →  ④ 인사평가(팀장·본부장·차순위임원 의견)  →  ⑤ 평가등급 확정(최상위 임원)
   →  ⑥ HR 일괄 공개  →  평가결과 확인
```

### 7.2 인사평가 (`인사평가`, 팀장·본부장·차순위임원)
- 팀원·팀장에 대한 **평가 의견 + 등급** 을 작성합니다(1차: 팀장, 2차: 본부장).
- 상세 패널: **등급 입력 → 자기평가 → 육성면담서** 순서. 개인 카드에 본인 자기평가 점수, (본부장의 경우) 팀장 등급의견도 함께 표시됩니다.

### 7.3 평가등급 확정 (`평가등급확정`, 임원 최상위)
- 소관 조직 팀장·팀원의 **등급을 최종 확정** 합니다.
- 개인 카드에 자기평가 점수, 팀장·본부장 등급의견을 함께 확인합니다.

### 7.4 조직평가관리 (`조직평가관리`, 최고관리자·HR마스터)
- 부문/공장 단위로 조직 평가등급을 결정합니다. (조직평가 단위는 부문으로 제한)

### 7.5 평가결과 확인 (`평가결과 확인`, 팀원·팀장)
- **HR이 일괄 공개한 이후** 본인 평가등급·조직등급·최종 평가권자의 의견을 확인합니다.
- 직전 2개년 포함 **최근 3년** 이력을 선택해 볼 수 있습니다.
- 팀장은 소속 팀원의 평가등급·평가의견도 확인 가능합니다.

### 7.6 전사 평가진행확인 (`전사 평가진행확인`, 최고관리자·HR마스터)
- 전사 평가 진행 상태를 한눈에 점검합니다.

> ### 🚨 인사평가 결과 가시성 원칙
> - 팀원은 본인 등급만, 팀장은 본인+책임 팀원, 본부장은 본부 산하, 임원은 책임 조직까지만 볼 수 있습니다.
> - 조직등급 쿼터 인원수는 일반 사용자에게 노출되지 않습니다.
> - 개인평가등급은 **HR 공개 이후에만** 노출됩니다.

## 8. AI 인사·성과 분석 (`AI 인사·성과 분석`, 최고관리자·HR마스터)

- 누적된 업무 실적(핵심목표·주간업무보고·자기평가·육성면담서·평가등급·혁신·포상)을 종합해 자유 질의에 답하는 **AI 챗봇** 입니다.
- 활용 예:
  - "전체 인원 기준 성과를 요약하고 서열을 매겨줘"
  - "회사 내 직무별 JD(직무기술서)를 만들어줘"
  - "핵심목표 완료율이 낮은 인원과 원인을 분석해줘"
- **사용법**: 분석 기간 선택(당해/최근 3년) → **데이터 준비** 클릭 → 질문 입력.
- 평가 기준(난도·완료·포기·자기평가·혁신·주간 추진내용의 효율성·실효성·중대성)과 **등급 정의(A=탁월 / B=우수 / C=보통 / D=미흡 / E=미달)** 는 AI 성과요약과 동일하게 적용됩니다.
- 답변은 **참고용** 이며 최종 판단은 사람이 합니다.
- 데이터 거버넌스: Google **Vertex AI** 백엔드를 사용하며, **입력 데이터를 모델 학습에 사용하지 않습니다.**

## 9. HR 관리 기능

### 9.1 기본정보입력
| 메뉴 | 기능 |
|------|------|
| **조직 관리** | 조직 트리(회사·부문/공장·본부·팀) 구성, 구성원 배정, 연도별 이력 |
| **사용자 관리** | 사용자 초대·수정, **엑셀 양식(이름·입사일 등) 일괄 등록** |
| **마일리지 관리** | 마일리지 조회·수정, **엑셀 양식 다운로드 → 입력 마일리지만 채워 일괄 업로드** |
| **포상 이력 관리** | 개인별 포상 이력 입력, 엑셀 업/다운로드 |
| **혁신활동 관리** | 스마트프로젝트·TDS 등 혁신활동 관리(마일리지 자동 집계), 엑셀 업/다운로드 |
| **연간 목표 관리** | 부문/공장(임원 업무목표) 연간 목표 입력 |

### 9.2 인사평가 설정 (HR마스터)
| 메뉴 | 기능 |
|------|------|
| **평가기간 관리** | 평가 시작·종료일, 결과 공개 시점 설정 |
| **평가이력 관리** | 3개년 탭 + 전년도 비교, Excel 다운로드 |
| **조직평가관리 / 조직평가인원관리** | 조직 등급 결정 / 등급별 인원 배정 |
| **개인평가등급 설정** | 개인평가 등급 체계 설정 |

### 9.3 시스템 설정
| 메뉴 | 기능 |
|------|------|
| **연도 전환 관리** | 당해 데이터를 전년 이력으로 이전하고 새 연도 세팅 |
| **데이터 백업 관리** | 전체 컬렉션 스냅샷 백업/복원(월요일 자동 백업), HR마스터만 삭제 |
| **HR 마스터 권한 관리** (최고관리자) | HR 마스터 권한 부여/회수 |
| **감사 로그** | 주요 액션 기록 조회(필터·Excel), 보안 스캔 |

## 10. 연도별 이력 관리
- 프로그램은 매년 **1월 1일 ~ 12월 31일** 이 하나의 사이클입니다.
- 조직관리를 제외한 대부분의 데이터(목표·평가·자기평가·육성면담서·주간보고 등)는 **연도별로 이력 관리** 됩니다.
- 상단 연도 선택으로 과거 연도 데이터를 조회할 수 있습니다.
